# 结构识图规则 V1.0

## 总说明与定位

- SR-001：先读取结构总说明、材料等级、抗震参数、保护层和钢筋符号约定，再读取构件图。
- SR-002：构件以“编号 + 楼层/标高 + 轴线范围”定位；编号相同但楼层或轴线不同的实例分别记录。
- SR-003：结构标高与建筑标高分开保存；存在建筑完成面与结构面差值时明确转换关系。

## 柱、梁、板

- SR-101：柱记录截面宽、截面深、层高、数量、混凝土等级和截面变化；变截面柱按标高分段。
- SR-102：梁记录净跨/轴线跨采用口径、截面、顶标高、数量和支座关系；悬挑段单独记录。
- SR-103：板按封闭边界、厚度、标高和混凝土等级分区；降板、洞口、后浇带和悬挑板作为边界事件。
- SR-104：梁板柱交接处的混凝土与模板计量边界必须在项目规则中统一，不在识图阶段隐式扣减。

## 基础与钢筋

- SR-201：基础记录类型、编号、平面尺寸、厚度/高度、顶底标高、垫层和混凝土等级；承台、基础梁和桩分开建模。
- SR-202：钢筋记录等级、直径、根数或间距、布置范围、层位、锚固、搭接、弯钩和来源。
- SR-203：平法标注、构件表和详图冲突时列出三方值；不要自动选择较大值或经验值。
- SR-204：钢筋长度缺少锚固、搭接或构件净尺寸时标记 `pending`；不得按经验补齐后进入重量计算。
