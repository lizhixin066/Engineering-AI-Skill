# Drawing Rules V1.0

## 输入控制

- DR-001：先确认专业。支持建筑、结构、给排水、暖通、电气、消防、钢结构、装修、园林、市政、道路、桥梁；无法判断时输出“专业无法识别”并停止该图计算。
- DR-002：建立图纸台账，至少记录图号、图名、版次、日期、比例、单位和有效状态。缺失项写入完整性异常。
- DR-003：优先采用图签比例和图纸标注单位；比例或单位冲突时停止比例推算。
- DR-004：仅在用户明确允许且没有任何尺寸标注时使用比例量取；结果标为 `inferred`，不得进入严格模式结果。
- DR-005：扫描图对关键文字进行两次独立 OCR 核验；结果不一致时标记人工确认。

## 空间与构件模型

- DR-101：建立轴网模型，记录横纵轴号、相邻轴距和总尺寸链；尺寸链不闭合时记录差值。
- DR-102：建立标高模型，记录 ±0.000、室内外、层面、基础、屋面与地下室标高；正负号和单位必须复核。
- DR-103：建筑图建立房间模型，记录房间名称、编号、用途、边界和面积来源。
- DR-104：墙体按剪力墙、砌体、轻质隔墙、防火墙等分类，记录厚度、材料、长度、标高范围和编号。
- DR-105：门窗按编号、宽高、数量、开启方向和位置提取，并与门窗表交叉核验。
- DR-106：结构构件按柱、梁、板、楼梯、基础分类；记录编号、截面、跨度/边界、标高、材料等级和配筋来源。
- DR-107：钢筋记录等级、直径、数量、间距、锚固、搭接和保护层；没有明确设计依据时不得补齐长度。

## 关联与置信度

- DR-201：通过构件编号、轴线和标高关联建筑、结构、机电和装修信息；名称相同但定位不一致的构件不得合并。
- DR-202：检查缺页、缺说明、缺详图、缺门窗表、缺结构总说明与缺钢筋说明。
- DR-203：对每条提取标记 `confirmed`、`inferred` 或 `pending`。`inferred` 必须说明推断路径；`pending` 必须说明缺失事实。
