# 建筑识图规则 V1.0

## 平面与房间

- AR-001：先确认建筑平面图的楼层、轴网方向、标高基准和有效版次，再识别房间与构件。
- AR-002：房间边界必须闭合；边界不闭合、跨防火分区或存在挑空时，面积标为 `pending`。
- AR-003：分别记录房间建筑面积、使用面积和净面积的口径；不要把轴线面积、墙中线面积与净面积混用。
- AR-004：同名房间按房间编号或“楼层 + 轴线范围”区分，不按名称直接合并。

## 墙体、门窗与洞口

- AR-101：墙体记录定位基准（轴线、中心线、内边线或外边线）、长度、墙高、厚度、材料和起止标高。
- AR-102：墙高优先从剖面、立面或做法表读取；不能仅用层高替代净墙高。
- AR-103：门窗同时读取平面编号与门窗表规格；编号、宽高、数量或防火等级不一致时生成冲突。
- AR-104：洞口按独立构件记录宽、高、数量、所属墙体和来源；不要在墙体净量中只保存扣减结果而丢失洞口明细。

## 楼梯、装修与关联

- AR-201：楼梯记录楼梯编号、跑数、踏步数、梯段宽、平台尺寸、层高和上下行方向；平面与剖面相互核对。
- AR-202：装修做法按房间、墙面、楼地面、顶棚分别关联；材料或做法编号不同的区域不得合并。
- AR-203：建筑门窗和洞口必须与结构洞口、消防门和装修收口建立关联；无法关联时保留问题项。
