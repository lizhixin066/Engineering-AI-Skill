# Review Rules V1.0

- RR-001：检查图纸版次、有效状态、单位和比例；被替代图纸不得作为最终量来源。
- RR-002：检查轴网尺寸链、标高链与构件总尺寸；差异必须定位到图号和轴线。
- RR-003：检查构件编号、材料等级、厚度、洞口和数量是否在平面、详图、表格间一致。
- RR-004：检查单位量级：mm/m 混淆、m²/m³ 混淆、kg/t 混淆均为阻断性异常。
- RR-005：检查重复计量边界，特别是梁板、墙柱交接、洞口、设备基础、后浇带和不同楼层。
- RR-006：按专业、楼层、分区和构件类型汇总并进行数量级审查；异常只报告，不自行篡改。
- RR-007：最终报告按 `confirmed`、`inferred`、`pending` 分组，保留每条异常的来源、影响范围和处理建议。
