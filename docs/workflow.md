# 工作流

1. 建立图纸台账：图号、专业、版次、日期、比例、单位及有效状态。
2. 识别专业、轴网、标高和构件；逐条保存图号/页码、轴线或构件编号、标注原文和置信状态。
3. 按材料、标高、厚度、部位和计量规则拆分构件，将图纸台账、轴网、标高、构件和 OCR 复核写入图纸模型。
4. 运行 `validate_drawing_model.py --strict`；先处理版本、尺寸链、OCR、低置信度、缺页和缺尺寸问题。
5. 只将通过校验的确认数据转换为算量 JSON 并运行计算器；未确认项禁止并入最终量。
6. 复核数量级、单位、尺寸链、洞口扣减和跨图一致性。
7. 交付图纸台账、图纸模型校验报告、分项计算书、汇总表、异常/假设清单及复核结论。

默认输出设计净量。仅在项目明确规定时，通过 `waste_factor`、工作面、放坡、搭接或其他可追溯参数计算施工/计价口径量。
