# 工作流

1. 建立图纸台账：图号、专业、版次、日期、比例、单位及有效状态。
2. 识别专业、轴网、标高和构件；逐条保存图号/页码、轴线或构件编号、标注原文和置信状态。
3. 按材料、标高、厚度、部位和计量规则拆分构件。先处理冲突、缺页和缺尺寸。
4. 将确认数据写入 JSON，运行计算器；未确认项保留在异常清单，禁止将其并入最终量。
5. 复核数量级、单位、尺寸链、洞口扣减和跨图一致性。
6. 交付图纸台账、分项计算书、汇总表、异常/假设清单及复核结论。

默认输出设计净量。仅在项目明确规定时，通过 `waste_factor`、工作面、放坡、搭接或其他可追溯参数计算施工/计价口径量。
